# Support

## Where To Get Help

Use GitHub Issues for:
- Reproducible bugs.
- Feature requests.
- Documentation problems.
- Build issues with clear environment details.

Before opening an issue, include:
- macOS version.
- Xcode version.
- Tuist version.
- Baseline version or commit.
- Relevant logs or screenshots.

## Not Covered

This project does not provide personal support for:
- General macOS troubleshooting.
- Homebrew account or installation problems unrelated to Baseline.
- App Store account issues.
- Unsigned app Gatekeeper policy decisions on managed devices.

## Security Issues

Do not report security issues in public support threads. Follow [SECURITY.md](SECURITY.md).
